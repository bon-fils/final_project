ID Number Character Counter Test

This demonstrates the real-time character counting functionality for the ID number field.

Must be exactly 16 characters long. 0/16
Test Instructions:
  1. Click in the ID number field above to focus it
  2. Type some characters (less than 16) and then click outside the field
  3. Notice the character counter appears showing "X/16" in red
  4. Try typing exactly 16 characters and click outside - counter disappears
  5. Try typing more than 16 characters and click outside - error message appears
  6. Click back in the field to clear the validation messages
Expected Behavior:

Validation only appears when you leave the field:

  • Empty field: No validation messages
  • 1-15 characters: "ID must be exactly 16 characters (X/16)" + counter shows
  • Exactly 16 characters: "✓ ID number is valid" + counter hidden
  • More than 16 characters: "ID must be exactly 16 characters - Too long!" + counter hidden
  • Focus back in field: All validation messages clear
